# Baitech Landing Page

Single-page React landing page for Baitech - AI-first Property Management for Bahrain & GCC.

## Features

- Single-page application with smooth scroll navigation
- All sections on one page (no routing)
- Responsive design (mobile-first)
- Brand colors and Filson Pro typography
- AOS animations
- Swiper carousel for app screens

## Tech Stack

- React 19.2.0
- TypeScript
- Vite
- Bootstrap 5.3.3
- AOS (Animate On Scroll)
- Swiper
- Phosphor Icons

## Brand Colors

- Light Blue: `#84DADE`
- Medium Blue: `#1ECAD3`
- Darker Blue: `#0099A8`
- Warm Red: `#FF4438`
- Dark Navy: `#002B49`

## Typography

- Primary Font: Filson Pro (loaded via Google Fonts)

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Sections

1. Hero - Main banner with CTAs
2. Trust Strip - Client logos
3. Features - 4 feature cards
4. About - Company information
5. Services - Service list
6. How It Works - 3-step process
7. App Screens - Carousel of app screenshots
8. Metrics - Animated counters
9. Pricing - 3 pricing tiers
10. For Landlords - Benefits for landlords
11. For Tenants - Benefits for tenants
12. Testimonials - Client testimonials
13. FAQ - Accordion FAQ section
14. Final CTA - Call-to-action section
15. Contact - Contact section (placeholder)
16. Footer - Footer with links

## Navigation

All navigation uses smooth scroll to anchor sections. No React Router is used.
